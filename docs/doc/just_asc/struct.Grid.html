Expand description

A Grid is what you draw on. Rather than concerning yourself with each character and how its position maps to your drawing, a Grid gives you a 100x100 canvas, with (0,0) in the top-left corner.

grid.line(0., 0., 100., 100.); // a line from top-left to bottom-right
grid.circle(50., 50., 25.); // a circle dead center
